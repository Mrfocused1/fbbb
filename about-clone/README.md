# FBBS Training Academy — About page

About page for the FBBS Training Academy site clone.

## How to view locally

```bash
npm run start:about
```

Then open http://localhost:4174 in your browser.

## What's included

- About page HTML with all internal assets served locally
- CSS/JS reused from the homepage-clone theme
- Images, SVGs, and other media assets
- Third-party resources (Google Fonts, analytics) remain remote

## Notes

- Navigation and footer match the site-wide nav (homepage-clone/about-clone/contact-clone)
- WordPress admin functionality has been removed
